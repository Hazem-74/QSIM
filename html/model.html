class model.Graph

Bases: MultiDiGraph

The graph class responsible for the computations and visualization of the optical system.

Inherits from networkx.MultiDiGraph

Variables:

  • counts (collections.defaultdict) – A dictionary that stores the counts of optical elements of each type (e.g. Laser, Beamsplitter, etc.).

  • elements (collections.defaultdict) – A dictionary that stores the optical elements by type (e.g. Laser, Beamsplitter, etc.).

  • path_modes_count (int) – stores the total number of distinct path modes within the optical system.
